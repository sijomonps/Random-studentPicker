# 🎯 Random Student Picker

A fun and simple website to randomly select students for classroom activities.

---

## 📌 About

In our class, teachers used random numbers to pick students, which was not very engaging.
So, I created this website that randomly selects a student from a list with fun prompts.

---

## ✨ Features

* 🎲 Random student selection
* 😂 Fun messages like “Picking the most creative student...”
* 📋 Preloaded list of all students in the class
* ⚡ Simple and fast

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 🎯 Purpose

To make classroom activities more interactive, fair, and enjoyable.
