A simple and fun web app to randomly pick a student from our class.
</p>

---

## 🌟 Why I Made This
We always used random number generators in class, which felt boring and not connected to us.  
So I created a small website that picks names directly from our batch list—simple, fun, and made just for us.  
AI tools helped me build and design it quickly.

---

## ✨ Features
- 🎲 Random student selection  
- 🎭 Fun loading animations  
- ⚡ Quick mode for instant results  
- 🖌️ Clean and modern design  
- 🎓 Hard-coded list for MCA 2025–2027  

---

## 📝 How it works 
1. Click **Pick a Student**  
2. Turn on **Quick Mode** if needed  
3. Wait for the result  

---

## 🛠️ Tech Used
- HTML  
- CSS  
- JavaScript
